# Tabs Keeper 🗂️

**Tabs Keeper** is a simple and time-saving Chrome extension that lets you save a group of tabs and reopen them all later with just one click. Perfect for developers, students, and multitaskers who want to quickly recreate their workspace.

---

## 🔧 Features

- Save a custom set of tab names and URLs
- Open all saved tabs with a single click
- Easy-to-use and lightweight interface
